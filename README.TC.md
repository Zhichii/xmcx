# XMCX 專案

## 簡介

**這個專案正在開發！**

這個項目包含三個部分：

- **CHH (C++ HillQiu's Helper)**: 這是一個輕量級的 C++ 工具箱，带有檔案、Windows 資源讀取和 JSON 处理工具。

- **HTI (HillQiu Text user Interface)**: 這是一個輕量級的 C++ 文字使用者介面框架。

  - **控制項**: 按鈕和列表。

- **XMCX**: 這是一個 Minecraft 啟動器。
