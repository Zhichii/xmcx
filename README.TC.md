# XMCX 項目

## 簡介

**這個項目正在開發！**

這個項目包含三個部分：

- **CHH (C++ HillQiu's Helper)**: 這是一個輕量級的 C++ 工具箱，带有文件、资源和 JSON 处理工具。

- **CLI (Command-Line Interface)**: 這是一個輕量級的 C++ “界面”框架，實際運行于於命令行界面中。

  - **控件**: 按鈕和列表。

- **XMCX**: 這是一個 Minecraft 啟動器。
