# XMCX 项目

## 简介

**这个项目正在开发！**

这个项目包含三个部分：

- **CHH (C++ HillQiu's Helper)**: 这是一个轻量级的 C++ 工具箱，带有文件、Windows 资源读取和 JSON 处理工具。

- **HTI (HillQiu Text user Interface)**: 这是一个轻量级的 C++ 文本用户界面框架。

  - **控件**: 按钮和列表。
  
- **XMCX**: 这是一个 Minecraft 启动器。
